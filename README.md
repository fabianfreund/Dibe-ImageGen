# DIBE - Imagine

DIBE - Imagine is a cross-platform application for AI image generation.

## Getting Started

### Prerequisites

- **Node.js** 18+ (LTS recommended)
- **npm** 9+

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the development server with hot-reload:
```bash
npm run dev
```

## Usage

To use DIBE - Imagine, you'll need a Gemini API key:
1. Obtain an API key from [Google AI Studio](https://makersuite.google.com/app/apikey).
2. Enter the key in the app's **Settings** page.
